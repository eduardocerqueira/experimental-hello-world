# Hello World in Multiple Languages

This repository contains minimal "Hello, World!" examples in multiple programming languages.

## How to Run Each Example

### Python
```bash
cd python
python hello.py
```

### Rust
```bash
cd rust
cargo run
```

### Go
```bash
cd go
go run hello.go
```

### JavaScript (Node)
```bash
cd javascript
node hello.js
```

### Ruby
```bash
cd ruby
ruby hello.rb
```

### Java
```bash
cd java
javac Hello.java
java Hello
```

### C
```bash
cd c
gcc hello.c -o hello
./hello
```

### PHP
```bash
cd php
php hello.php
```

### Kotlin
```bash
cd kotlin
kotlinc hello.kt -include-runtime -d hello.jar
java -jar hello.jar
```

### Zig
```bash
cd zig
zig run hello.zig
```
